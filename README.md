# Netflix Clone

A full-stack Netflix clone application built with Spring Boot for the backend. This project aims to replicate core Netflix functionalities such as user authentication, video streaming, and watchlist management.

## Features

*   **User Authentication**: Secure user registration, login, email verification, and password reset.
*   **Role-Based Authorization**: Differentiates between user and admin roles with protected routes.
*   **Video Management**: Admins can upload, update, and manage video content.
*   **Video Streaming**: Efficient video streaming with support for range requests (partial content).
*   **Watchlist**: Users can add and remove videos from their personal watchlist.
*   **File Uploads**: Secure storage for video and image files.
*   **JWT Authentication**: Token-based security for API access.
*   **Email Service**: Integration for sending verification and password reset emails.

## Technologies Used

### Backend (Spring Boot)

*   **Java 17+**
*   **Spring Boot**: Framework for building robust, production-ready applications.
*   **Spring Data JPA**: For database interaction and repository management.
*   **Spring Security**: For authentication and authorization (JWT-based).
*   **MySQL**: Relational database for storing application data.
*   **JJWT**: Java JWT library for creating and verifying JSON Web Tokens.
*   **Jakarta Mail**: For sending emails.
